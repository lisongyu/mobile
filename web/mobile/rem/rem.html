说明

W3C官网描述rem为：

“font size of the root element”。

以IPhone5的分辨率为基准设定1rem = 20px

通过获取不同设备的文档宽度

计算其于基准分辨率的比例关系

动态设定1rem的值，实现适应不同设备。

    $(function(){
        var oHtml=$('html');
        var phptoWid='';
        var currentRem='';
        function setPx(){
            phptoWid=oHtml.width();
            currentRem=phptoWid/320*20+'px';
            oHtml.css('fontSize',currentRem);
        }
        setPx();

        $(window).on('resize orientationchange').resize(function(){
            setPx();
        });
    });
